# Tags

This page lists all tags used throughout the Discovering Ben research site. Tags help organize content and enable filtering by audience, topic, and content type.

## By Audience

**autistic-users**
:   Content specifically relevant to autistic individuals using AI assistants

**researchers**
:   Academic and scientific research perspectives

**clinicians**
:   Clinical practice and assessment resources

**developers**
:   LLM developers and AI safety researchers

## By Cycle

**information-overload**
:   Cycle 1: Information Overload pattern

**decision-paralysis**
:   Cycle 2: The "One Best Thing" Paradox

**perfectionism**
:   Cycle 3: Perfectionism Spiral

**emotional-dysregulation**
:   Cycle 4: Emotional Dysregulation

**mind-reading**
:   Cycle 5: Mind Reading Expectations

**special-interests**
:   Cycle 7: Special Interest Engagement

## By Content Type

**case-study**
:   Real-world examples and case studies

**methodology**
:   Research methodology and approach

**intervention**
:   Practical interventions and solutions

**technical-analysis**
:   Technical LLM behavior analysis

## By Severity

**pathological-cycle**
:   Severe patterns requiring intervention

**mild-cycle**
:   Minor patterns with low impact

**natural-trait**
:   Healthy autism traits (not dysfunction)

---

## Usage

Tags are automatically indexed by the MkDocs tags plugin. Click any tag throughout the site to see related content.
