## Methodology for Deep Dives

### Standard Deep Dive Protocol

**For Each Investigation:**

1. **Filter & Scope**
   - Identify relevant conversations via keyword/category filters
   - Chronologically order to see evolution
   - Sample conversations for pattern validation

2. **Extract & Structure**
   - Pull key quotes, documents, lists, frameworks
   - Build structured data (tables, JSON, markdown)
   - Cross-reference related conversations

3. **Analyze & Synthesize**
   - Identify patterns and anomalies
   - Build comprehensive documentation
   - Extract reusable frameworks

4. **Validate & Refine**
   - Check for contradictions
   - Verify completeness
   - Test applicability

5. **Document & Deliver**
   - Create standalone deliverable
   - Provide context and usage guide
   - Note limitations and assumptions
